# Sistema de Gerenciamento de PetShop

## 📌 Descrição
Sistema web simples para gerenciamento de clientes e pets, desenvolvido para fins acadêmicos.

## 🚀 Funcionalidades
- Cadastro de clientes
- Cadastro de pets
- Visualização de clientes
- Visualização de pets
- Armazenamento local (LocalStorage)

## 🧪 Iteração 2
Nesta versão foram realizadas as seguintes melhorias:
- Criação da página pets.html
- Separação da visualização de clientes e pets
- Melhorias na interface
- Uso de controle de versão com Git (branch, merge e tags)

## 🛠 Tecnologias utilizadas
- HTML
- CSS
- JavaScript
