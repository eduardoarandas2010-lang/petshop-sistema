# Sistema de Gerenciamento de PetShop

## 📌 Descrição
Sistema web simples para gerenciamento de clientes e pets, desenvolvido para fins acadêmicos.

## 🚀 Funcionalidades
- Cadastro de clientes
- Cadastro de pets
- Visualização de clientes
- Visualização de pets
- Armazenamento local (LocalStorage)

## 🧪 Iteração 2
Nesta versão foram realizadas as seguintes melhorias:
- Criação da página pets.html
- Separação da visualização de clientes e pets
- Melhorias na interface
- Uso de controle de versão com Git (branch, merge e tags)

## 🛠 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 💻 Como executar o projeto

1. Baixe ou clone o repositório:
git clone https://github.com/eduardoarandas2010-lang/petshop-sistema.git

2. Abra a pasta do projeto

3. Execute o sistema abrindo o arquivo:
index.html

4. O sistema abrirá no navegador e estará pronto para uso

## 📂 Estrutura do projeto
- index.html → página inicial
- clientes.html → exibição de clientes
- pets.html → exibição de pets
- style.css → estilos
- script.js → lógica do sistema
